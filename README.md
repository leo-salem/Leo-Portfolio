# My portfolio

Source for my personal site. I'm Mohamed Salem — fresh CS grad from Menofia University, Java backend dev based in Cairo.

It's plain HTML and CSS, no framework, no build step. The reason: I read a survey of 60+ hiring managers that basically said "stop wasting time on portfolio websites, just make something that loads fast and tells your story." That stuck. The whole thing is around 20 KB before the photo.

## Run it locally

Open `index.html` in any browser. That's the entire setup.

```
start index.html
```
